Sistem Monitoring Energi

Sistem monitoring berbasis Laravel & ESP32 yang menampilkan data konsumsi energi secara real-time. Menggunakan sensor PZEM-004T untuk membaca tegangan, arus, dan daya, lalu mengirimkannya ke server via HTTP/REST API.

Fitur utama

  • Monitoring arus, tegangan, dan daya real-time
  • Grafik historis dan export data
  • Notifikasi bila konsumsi melewati ambang batas
  • Autentikasi pengguna dan role-based access

Teknologi

Laravel, MySQL, ESP32, PZEM-004T, json, REST API
